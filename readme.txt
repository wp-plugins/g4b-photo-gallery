A simple yet powerful photo gallery plugin for Wordpress.


== Description ==

G4B Photo Gallery is a free, flexible plugin for Wordpress. It provides a simple and effective way to 
include a complete photo album anywhere on you blog, just by typing in a keyword.

[Live Demo](http://geekforbrains.com/wordpress/g4b-photo-gallery-plugin-for-wordpress/ "Live demo of G4B Photo Gallery")

Plugin Features:

* Easily customize with CSS
* Manage and create albums
* Upload, edit and delete photos
* Set thumbnail and full view sizes
* Supports cropping both thumbnail and full view
* Supports PHP 4 and PHP 5


== Installation ==

1. Unzip and upload the G4B_gallery plugin folder to wp-content/plugins/
2. Make sure that the folder wp-content/uploads/ exists and is writable by the server (CHMOD 755)
3. Activate the plugin in your Wordpress Admin panel, under the Plugins page.

Usage:

To embed your gallery on any Wordpress page or post, simply type in the tag: {g4b_gallery}

When the page content is loaded, the plugin looks for this tag, and swaps it out dynamically
with your photo album.

If you would like to embed the gallery with PHP, you can use this function call:

`<?php G4B_gallery_view(); ?>`

This function does NOT have to be in the Wordpress loop.



== Frequently Asked Questions ==

= Can I set the size of images created? =

Yes. These options are available in the G4B Gallery options page located in your Wordpress admin.

= Does it support PHP 5? =

Yes. G4B Photo Gallery supports PHP 4 and PHP 5.

= What image file types are supported? =

G4B Gallery supports JPG, GIF and PNG images.
